# Kota Doria Wholesale Only — Product Catalog

A premium textile product catalog with an admin panel for uploading and managing products.

## Features
- Browse products by category (Embroidery, Block Print, Brush Paint, Screen Print)
- Multi-image slider per product
- Download individual product images
- Admin panel to upload new products and delete existing ones
- Indian Rupee (₹) pricing

## How to Run Locally

1. **Install Node.js** (if not already installed): https://nodejs.org/
2. **Open terminal** in this folder
3. **Install dependencies:**
   ```
   npm install
   ```
4. **Start the server:**
   ```
   node server.js
   ```
5. **Open in browser:** http://localhost:3000

## Admin Panel
Navigate to: http://localhost:3000/admin.html
